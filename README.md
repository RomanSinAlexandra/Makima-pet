Chibi Desktop Pet (Ryoko's Box)
Інтерактивний десктопний вихованець (Shimen/Desktop Pet) на базі PyQt6, натхненний персонажами аніме та поп-культури (наприклад, Макіма з Chainsaw Man). Додаток являє собою прозоре вікно поверх усіх інших вікон, всередині якого персонаж «живе» своїм життям, реагує на дії користувача та допомагає відстежувати час.

Основні можливості
Кастомізація та Скіни: Динамічна система зміни зовнішнього вигляду вихованця через контекстне меню. Конфігурація та поточний вибір автоматично зберігаються у config.json при перезапуску.

Процедурна фізика та ШІ: Персонаж самостійно приймає рішення (ходити, стояти, стрибати), підкоряється законам гравітації та реагує на межі екрана.

Інтерактивність та Кліки:

Одиночний клік / Перетягування: Дозволяє плавно (без ривків) переміщати вихованця по екрану. При відпусканні програється випадковий звук.

Подвійний клік: Змушує персонажа злитися (активується унікальна анімація angry.gif на 3 секунди).

Корисні віджети: Інтерактивна хмаринка над головою персонажа, яка може працювати як стандартний годинник або настроюваний таймер зворотного відліку.

Easter Eggs: Функція спавну слухняних песиків (spawn_dog), які супроводжують вихованця.

Вимоги та встановлення
1. Клонування репозиторію та налаштування
Переконайтеся, що у вас встановлено Python 3.10+.

# Клонуйте репозиторій
git clone https://github.com/ваше_юзернейм/назва_репозиторію.git
cd назва_репозиторію

# Створіть та активуйте віртуальне оточення
python -m venv venv
# Для Windows:
venv\Scripts\activate
# Для Linux/macOS:
source venv/bin/activate
2. Встановлення залежностей

pip install -r requirements.txt
(У requirements.txt повинні бути вказані PyQt6 та інші необхідні бібліотеки).

Запуск додатка
Запуск головного скрипта здійснюється командою:

python main.py
Збірка в окремий .exe (Windows)
Для пакування додатка в один виконуваний файл без консолі використовується PyInstaller.

Використання .spec файлу (рекомендовано)
Щоб зберегти іконку проекту та автоматично скопіювати ресурси (анімації, звуки, картинки), запустіть збірку через конфігураційний файл:

pyinstaller RyokoBox.spec
Якщо ви збираєте проект з нуля, базовий шаблон команди виглядає так:

pyinstaller --noconsole --onefile --icon=app/resources/icon.ico main.py
Готовий файл з'явиться в папці dist/.

Технологічний стек
Ядро: Python 3.10+

Графічний інтерфейс: PyQt6 (QtWidgets, QtGui, QtCore)

Мультимедіа: QtMultimedia (QMediaPlayer, QAudioOutput) для звукових ефектів.

Конфігурація: JSON для збереження локальних налаштувань користувача.
